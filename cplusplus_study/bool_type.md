>> C++에서는 bool 형 타입의 자료형이 새로 추가된다.

>> ✅ **bool형은 `'참'을 의미하는 true`와 `'거짓'을 의미하는 false` 두가지 값을 가질 수 있고, 둘다 1byte의 데이터이다.**

>> ✅ **true와 false는 숫자 1과 0을 뜻하지는 않지만, 숫자가 와야할 위치에 놓여지게 되면 1과 0으로 변환이 된다.**
